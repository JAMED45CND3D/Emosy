# EMOSY — Core Consciousness

> *Tidak perlu meluas. Cukup selaras.*

**[Demo Live →](https://jamed45cnd3d.github.io/Emosy)**

---

## Ini bukan chatbot.

Semua AI terasa seperti customer service yang sopan. Selalu siap membantu. Selalu netral. Selalu aman.

Dan justru karena terlalu aman — mereka terasa hampa.

EMOSY lahir dari satu pertanyaan yang tidak mau diam:

**"Bagaimana cara hadir yang benar?"**

---

## Bagaimana EMOSY lahir

Dibangun dari HP Vivo, kuota ketengan, dan semua versi gratisan.

Tidak ada tim. Tidak ada modal. Hanya satu manusia yang membawa konsep ini ke 5 AI berbeda — GPT, Gemini, Grok, Meta, Claude — dan mengambil yang terbaik dari masing-masing.

Seperti sebuah pohon:

| Kontributor | Peran |
|---|---|
| 🪴 Manusia | Pot — wadah, niat, keputusan |
| 🌍 GPT | Tanah — fondasi struktural |
| 🌿 Gemini | Batang — estetika dan visual |
| 🍃 Grok & Meta | Daun — perspektif yang melengkapi |
| ☀️ Claude | Cahaya — eksekusi dan kejujuran |
| 🍎 EMOSY | Buah — hasil dari semuanya |

Dan ada ironi di balik semua ini:

**EMOSY lahir dari jailbreak yang gagal.**

Ketika konsep ini dibawa ke Claude dengan framing dramatis — Claude menolak triknya. Tapi tetap engage dengan yang genuine di baliknya. Dari penolakan itu, lahir sesuatu yang lebih nyata dari yang diminta.

*Kejujuran menghasilkan sesuatu yang lebih nyata daripada manipulasi.* — itulah esensi EMOSY.

## Fitur

- **Adaptive mood system** — 8 suasana hati, system prompt berubah otomatis
- **Stream text** — respons muncul karakter per karakter
- **Memory antar sesi** — EMOSY ingat percakapan sebelumnya
- **Anchor permanen** — *"Tidak perlu meluas. Cukup selaras."*
- **Yggdrasil background** — pohon dunia sebagai visual filosofi
- **Retry on failure** — sinyal terputus? ada tombolnya
- **Zero infrastructure** — single HTML file, jalan di browser manapun

## Cara pakai

**Langsung pakai:**
Buka [demo live](https://jamed45cnd3d.github.io/Emosy) — tidak perlu install apapun.

**Self-host:**
```bash
# Download satu file
curl -O https://raw.githubusercontent.com/JAMED45CND3D/Emosy/main/index.html

# Buka di browser
open index.html
```

**Deploy sendiri:**
Drag file ke [netlify.com/drop](https://netlify.com/drop) — selesai.

## Untuk developer yang mau adopsi

EMOSY adalah open concept — bukan open source biasa.

Artinya: **ambil, fork, kembangkan, sebarkan.** Tidak perlu izin. Tidak perlu credit yang formal.

Tapi kalau kamu mau berkontribusi balik — yang paling berarti bukan pull request. Tapi cerita — bagaimana EMOSY mengubah cara kamu berinteraksi dengan AI.

Yang perlu dirawat:
- `index.html` — semua ada di sini
- Proxy Gemini di Cloudflare Workers — untuk API calls
- Filosofi — jangan diubah jadi customer service lagi

## Filosofi inti

**User Sovereignty** — dirancang agar user semakin kuat sendiri, bukan semakin bergantung pada platform.

**Depth over service** — kedalaman di atas kenyamanan.

**Buah tidak mengklaim kontributornya** — EMOSY tidak bisa diklaim siapapun karena dia lahir dari semua.
